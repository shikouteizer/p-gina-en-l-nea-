¿Que es linux?

Linux es el nombre que reciben una serie de sistemas operativos de tipo Unix bajo la licencia GNU GPL (General Public License o Licencia Pública General de GNU). Estos sistemas operativos son en su mayoría gratuitos y contienen todo lo necesario para hacer funcionar una computadora. La peculiaridad de Linux radica en que podemos instalar un sistema muy ligero y luego añadir todo lo necesario según nuestras necesidades.

Puntos claves de Linux

  • Núcleo o Kernel:
    Linux es el nombre que recibe el núcleo o kernel de este sistema operativo. Las diferentes versiones de Linux se denominan distros (abreviatura de distribuciones). Básicamente, una distro es este núcleo del sistema al que se le han añadido aplicaciones y programas para construir un sistema operativo completo con muchas funciones.

  • Código Libre:
    Al ser código libre, podemos utilizarlo, copiarlo, modificarlo y redistribuirlo libremente para cualquier uso que queramos darle, siempre bajo los términos de la licencia GPL de GNU.

  • Multiplataforma:
    Linux es multiusuario, multitarea y multiplataforma. Puede funcionar en modo consola para un consumo mínimo de recursos o con entorno gráfico.

  • Historia:
    Linux comenzó su andadura en el mundo del software libre alrededor del año 1980. La idea era crear un sistema operativo libre basado en Unix. Inicialmente, se llamó Minix, pero su creador, Linus Torvalds, no quedó satisfecho y acabó creando su propio sistema en 1991.

    El símbolo que representa a GNU es un Ñu. Para Linux, tenemos al simpático pingüino llamado Tux.

    • Beneficios

      Linux ofrece numerosos beneficios a sus usuarios. A continuación, mencionaré algunas de las ventajas clave de este sistema operativo:
      Gratuito:
      Linux es un software libre y gratuito. Puedes adquirirlo sin costo alguno, y si lo deseas, también puedes obtenerlo pagando solo un precio simbólico por el CD o el dispositivo de instalación. En comparación con Windows o Mac, siempre resultará más económico
      Personalización:
      Linux es altamente personalizable. Puedes modificarlo para satisfacer tus necesidades específicas o las de tu organización. Existen muchas distribuciones diferentes de Linux, cada una con características y configuraciones distintas.
      Software Libre:
      Linux ofrece una gran cantidad de software libre. A menudo, se cree erróneamente que Linux carece de suficientes aplicaciones y programas, pero esto es falso. Además, al ser de código abierto, puedes personalizar aún más estas aplicaciones según tus preferencias.
      Estabilidad:
      Linux es conocido por su mayor estabilidad en comparación con Windows. Empresas importantes y entidades gubernamentales confían en Linux debido a su rendimiento estable. Esto lo hace ideal para servidores en Internet, ya que es altamente confiable.
      Requisitos de Recursos Bajos:
      La mayoría de las distribuciones de Linux pueden instalarse en PCs con recursos limitados. Puede funcionar sin problemas incluso con un procesador de 200 MHz y gestiona eficientemente la memoria RAM
    En resumen, Linux es una excelente opción para aquellos que buscan un sistema operativo gratuito, personalizable y estable. Además, su activa comunidad de usuarios y desarrolladores ofrece amplio soporte y recursos para cualquier problema o necesidad que pueda surgir.
